## Backup Instructions

Since the source code is only available on Vercel and not connected to Git, you have two options:

### Option 1: Download from Vercel Dashboard
1. Go to https://vercel.com/venture-works/quantumenergybeds-vercel
2. Click on the latest deployment
3. Download the source files
4. Extract to this repository
5. Push to GitHub

### Option 2: Use Vercel CLI
```bash
# Install Vercel CLI
npm i -g vercel

# Login
vercel login

# Pull the project
vercel pull quantumenergybeds-vercel

# Copy files to this repo and push
```
